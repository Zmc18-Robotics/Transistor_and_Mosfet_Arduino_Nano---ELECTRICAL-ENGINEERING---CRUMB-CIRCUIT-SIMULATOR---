# Description

A project that combine the transistor and mosfet in Arduino Nano. Electrical systems, Hardware connections, and Logic

Controllable LED with Transistor and mosfet, Arduino Nano

Here the materials :
- Arduino Nano x1
- Transistor NPN x2
- Mosfet N x2
- Push button x4
- Led pin x4
- Buzzer x2
- Resistor 220Ω x7
- Resistor 1000Ω x4
